# Portfolio Description

**ML gRPC Prediction Service** is a compact backend portfolio project showing how an ML model can be exposed through a typed gRPC API.

The project includes a Protocol Buffers service definition, Python gRPC server, Python client, model-loading logic, Dockerfile, environment-based configuration and basic tests. It is intentionally lightweight so that reviewers can understand the full service design quickly.

## Skills demonstrated

- Python backend development
- gRPC and Protocol Buffers
- API contract design
- ML model serving pattern
- Docker-based deployment
- Environment-based configuration
- Client/server architecture
- Basic error handling and health checks
- Reproducible project structure
